1. `Цель`
Реализовать базовую шахматную доску и две фигуры — Конь (Knight) и Пешка (Pawn) — с возможностью перемещения фигур на доске по правилам шахмат.

2. `Основная логика`
    * Доска хранит фигуры в массиве $cells[x][y].
    * Фигура сама проверяет правила хода.

3. `Правила для фигур`
   *  Конь - L-образное движение (2 по x + 1 по y или 1 по x + 2 по y)
   * Правила хода: вперед на 1 клетку (только по y), без взятия диагональю

``` Для тестов можно использовать скрипт ChessCommand ```

``` Запуск проекта chmod +x install.sh и sail up -d ```
