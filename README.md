1. `Цель`
Реализовать базовую шахматную доску и две фигуры — Конь (Knight) и Пешка (Pawn) — с возможностью перемещения фигур на доске по правилам шахмат.


2. `Структура проекта`
   3. Классы
      * Board
        * Хранит состояние доски (клетки и фигуры)
        * Методы:
          * placePiece(ChessPiece $piece, Position $position): void — ставит фигуру на клетку
          * movePiece(Position $from, Position $to): void — перемещает фигуру с одной клетки на другую
          * getPiece(Position $position): ?ChessPiece — возвращает фигуру с клетки, либо null
          * setUp(): void — устанавливает стартовую расстановку фигур (например, только Knight и Pawn)
      * Position
        * Хранит координаты клетки: x и y
      * ChessPiece (интерфейс)
        * Методы:
          * move(Position $from, Position $to, Board $board): bool — проверяет возможность хода и при успехе перемещает фигуру
          * symbol(): string — возвращает символ фигуры (K для коня, P для пешки)


3. `Основная логика`
    * Доска хранит фигуры в массиве $cells[x][y].
    * Фигура сама проверяет правила хода.
    * Фигура вызывает Board->movePiece(), чтобы переместить себя.
    * Доска не проверяет правила шахмат, она просто хранит состояние

4. `Правила для фигур`
   *  Конь - L-образное движение (2 по x + 1 по y или 1 по x + 2 по y)
   * Правила хода: вперед на 1 клетку (только по y), без взятия диагональю
